## О проекте

Mesto — учебный проект по курсу Яндрекс Практикума "Ассоциированные программы: frontend-разработчик" о размещении фотографий красивых мест. Пользователь может редактировать профиль и аватар, добавлять и удалять карточки, ставить лайки, просматривать изображение в модальном окне. Интерфейс адаптивный, стили организованы по БЭМ.

### Основные возможности
- Редактирование имени и описания профиля, обновление аватара
- Добавление новой карточки (название + ссылка на изображение)
- Предпросмотр изображения в модальном окне
- Лайки карточек с отображением счётчика
- Удаление собственных карточек с подтверждением
- Закрытие модалок по кнопке, клику по оверлею и клавише Escape
- Клиентская валидация форм (обрезка пробелов, проверка на наличие хотя бы одной буквы)
- Просмотр статистик карточек

### Технологии
- HTML5, CSS3 (БЭМ, адаптивная вёрстка)
- Ванильный JavaScript (модули ES)
- Vite (dev‑сервер и сборка)
- GitHub Pages (публикация)

## Команды

- Локальный сервер (разработка):
  ```bash
  npm run dev
  ```
- Сборка проекта:
  ```bash
  npm run build
  ```
- Предпросмотр сборки:
  ```bash
  npm run preview
  ```
- Публикация на GitHub Pages:
  ```bash
  npm run deploy
  ```
